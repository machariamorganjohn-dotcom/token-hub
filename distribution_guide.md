# Token Hub Distribution Guide

Your permanent app files are now ready! This guide explains where to find them and how to install them.

## 1. Android App (APK)
The Android installation file is located here:
- **Path**: `build/app/outputs/flutter-apk/app-release.apk`
- **How to install**:
  - Transfer this `.apk` file to your Android phone.
  - Open the file on your phone and follow the prompts to install (you may need to allow "Install from Unknown Sources").

## 2. Web App (Browser)
The files for the web version are located here:
- **Path**: `build/web/`
- **How to use**:
  - You can upload these files to any web host (like GitHub Pages, Firebase, or Netlify).
  - Once uploaded, anyone with the link can use the app in their browser.

## 3. Windows Desktop
You can continue to use the `run_token_hub.bat` file in the project folder for fast desktop access.
